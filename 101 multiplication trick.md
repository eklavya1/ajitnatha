## 101 mult with 2 digits
- 101 x ab = abab

ex -
101 x 24 = 2424
101 x 50 = 5050
101 x 65 = 6565
## 101 mult with 3 digits
- 101 x abc = (a+abc)bc
- ex -
- 101 x 236 = (2+236)36 = 23836
